This paper describes the aggregation and ... of movie data from different online movie data sources to a unifed, linked open data set.
Therefor, the IMDb, TMDb, OFDb and Freebase are used.
Because one movie can exists in multiple of these data sources, the system has to recognize those movies and merge them to one entity.
Entities have different representation in differen data sources, that's why a matching algorithm has to be developed.
In this paper a matching algorithm is explained and evaluated.
The evaluation shows, that ...
Besides the system updates all movies from time to time, so that the resulting data set is always up-to-date.
